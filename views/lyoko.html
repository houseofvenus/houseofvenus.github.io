lyoko

Lyoko is the code behind Rooms's omnichannel programming environment. Using whatever inputs are available to a machine it can create a decentralized immersive application or DIA that can interface with any output available to a given user on the network. It can be thought of as a natural language programming language that also translates the human motion capture and other machine processed sensory data for Rooms to build immersive experiences.
